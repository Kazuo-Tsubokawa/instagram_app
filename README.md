# instagram-app
インスタグラムみたいに八幡対のおもひでの写真を投稿できるアプリ

## 概要
・写真の復数アップロード可のCRUD機能付きアプリです
・画像ファイルはFirebase上に保存
・GoogleFontを使用して、ロゴを装飾

## テーブル
